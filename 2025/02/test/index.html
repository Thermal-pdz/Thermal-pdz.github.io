装饰你的shell
装饰你的Shell安装软件MacOS的默认shell不太好看，我们安装一个oh-my-zsh对其进行装饰。先在自己的home目录新建一个专门用来放软件的文件夹 1mkdir -p ~/softwares/omz && cd ~/softwares/omz 然后，安装oh-my-zsh软件： 1sh -c "$(curl -fsSL https://gitee.com/pocmon/ohmyzsh/raw/master/tools/install.sh)" 修改主题若不喜欢默认的主题，可以在.zshrc文件中修改oh-my-zsh的主题。修改主题就是修改ZSH_THEME这个参数。re5et主题是Bensz觉得喜欢，我也觉得很不错的主题。你可以自己去github上看看喜欢哪款主题。 更新软件需要更新oh-my-zsh时，输入如下命令进行 1omz update
